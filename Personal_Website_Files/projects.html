Path of Ants

"Path of Ants" is a fantastic Android Application that challenges your eye to hand coordination, as well as your decision making process. It contains a simple concept where you have to save the ant from entering the restricted space by making sure that the ant stays on the designated path. As time passes by the ant runs faster and faster. Challenge your friends, and save the ants ! Download from Google Play Store today!

Zombie Apocalypse

"Zombie Apocalypse" is an amusing 2D Java Applet based game, which depicts the famous character "Megaman" facing off zombies. This game utilizes numerous sets of sprites, double buffering, and the JApplet API. This game was made on the Eclipse IDE, and this game can be run on any browser since this game is made on the JApplet API.
